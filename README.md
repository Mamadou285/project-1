# Coffee Interactive Story

## Live Demo
https://mamadou285.github.io/coffee-story/

## Project Overview
This project is an interactive coffee ordering story.

The user makes three choices:
- Hot or iced coffee
- Sugar or no sugar
- Pay with card or cash

The page updates based on the user's selection.

## Features
- Interactive buttons
- Story text updates
- Simple animation
- Responsive layout

## Technologies Used
HTML  
CSS  
JavaScript  
GitHub Pages for hosting
